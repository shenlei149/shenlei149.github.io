> 《古诗词课》，叶嘉莹，生活读书新知三联书店，2018 年版，2025 年印刷。

这本书从诗经开始介绍，一直讲到了南宋末年的词人王沂孙。全书分为两个部分，上编为诗，下编为词。共计 36 讲，涉及了几十位诗人和词人，内容丰富，讲解细致。每一讲都介绍了作者的生平背景、创作风格，分析其代表作品，并且结合历史背景和文化背景进行解读。

《诗经》是我国最早的一部诗歌总集。这些诗歌分为风、雅、颂三类，风指的是国风，各地的民间歌谣，雅分为大雅和小雅，除了民间歌谣还有贵族作品，颂是用于宗庙祭祀的诗歌。赋比兴是《诗经》中的三种基本表现手法。赋有铺陈之意，比有拟喻之意，兴有感发兴起之意。
```
苕之华，芸其黄矣。心之忧矣，维其伤矣。
苕之华，其叶青青。知我如此，不如无生。
牂羊坟首，三星在罶。人可以食，鲜可以饱。
```
苕，这里应该念 tiao，二声，指的是一种植物，指凌霄花或水草。到了秋天要凋落的时候就完全变成黄色了。

第一句是见物起兴，属于兴的变现手法。这里是由物及心。心和物之间的感应是微妙的，虽然这种作用之中必然有某种感性的关联，但并非每一种感性的关联都可以做理性的分析。它可以是完全凭直觉的触引，并不一定有理性的思索安排。

牂，zang，一声。牂羊是母羊，坟是大的意思，说明羊很瘦。罶，liu，三声，打鱼的竹筐。水中倒影着星星，说明鱼很少。这里是赋，直接描述。

```
硕鼠硕鼠，无食我黍！三岁贯女，莫我肯顾。
逝将去女，适彼乐土。乐土乐土，爰得我所。

硕鼠硕鼠，无食我麦！三岁贯女，莫我肯德。
逝将去女，适彼乐国。乐国乐国，爰得我直。

硕鼠硕鼠，无食我苗！三岁贯女，莫我肯劳。
逝将去女，适彼乐郊。乐郊乐郊，谁之永号。
```
女同汝。这里用的比的手法。作者心中痛恨那些一味剥削重敛、不顾百姓死活的家伙，不过不能明说，只能用硕鼠来打个比方。比和兴不同，是由心及物，心理已经有了某种情意，找一个外物来做比喻，写出心中的情意。这种表现形式是有理性思索安排的。

```
将仲子兮，无踰我里，无折我树杞。
岂敢爱之，畏我父母。
仲可怀也，父母之言，亦可畏也。

将仲子兮，无踰我墙，无折我树桑。
岂敢爱之，畏我诸兄。
仲可怀也，诸兄之言，亦可畏也。

将仲子兮，无踰我园，无折我树檀。
岂敢爱之，畏人之多言。
仲可怀也，人之多言，亦可畏也。
```
这首诗描述的是女子对所爱男子的叮咛和告诫，直接描述了内心的感情。这种手法就是赋。在传统中，直接描述可以是物，但不仅仅局限于目之所见的具体的外物，也可以是抽象的内心的情感，诗所描述的形象本身就是作者心中的情意。

总结赋比兴三者：赋是直接叙写，即物即心；比是借物为喻，心在物前；兴是因物起兴，物在心前。

《离骚》是《诗经》之后的又一巅峰。作者屈原名平，战国时期的楚国人。司马迁在《史记》中说他因“信而见疑，忠而被谤”，才写了《离骚》抒发自己的情怀，司马迁评价是“推此志也，虽与日月争光可也”。

屈原在《离骚》中反复陈述了希望楚国美好强盛的愿望，在愿望不能实现的苦闷之中，也想过退而自保，独善其身，但是他仍然不肯放弃自己的愿望，写出“既莫足与美政兮，吾将从彭咸之所居！”。屈原所追求的理想是最高远、最完美的，因此也是最难以达到的，但正是由于人类有这样的追求，所以人类才有希望。

中国的诗词，经常表现的一个主题就是对美好的事物、美好的对象、美好的理想的追求和怀思。这个传统，就是从屈原的《离骚》开始的。“美人香草以喻君子”，是中国传统文化中一个独特的比兴方法，源头来自《离骚》。悲秋的主题也是从这里开始传承。

《古诗十九首》是东汉时期一些无名氏的作品，非常易懂又相当难解，艺术成就极高却又没有留下作者的姓名，人人读了心中都觉得有所触动，却又很难说清缘由。《古诗十九首》在写作态度上十分真挚诚恳，语言也相当平易近人，但经受住了千百年来无数读者的反复挖掘，使每个人都够有所得有所感。内容涉及很多人生中常常遇到的问题，写出了人们心灵深处最普通也最深刻的几种感情的基本类型。这些感情来源于人与人以及人与社会之间的关系，凡是社会中的人，都不可能没有经历过。

下面是《东城高且长》这首诗。
```
东城高且长，逶迤自相属。
回风动地起，秋草萋已绿。
四时更变化，岁暮一何速！
晨风怀苦心，蟋蟀伤局促。
荡涤放情志，何为自结束？
燕赵多佳人，美者颜如玉。
被服罗裳衣，当户理清曲。
音响一何悲！弦急知柱促。
驰情整巾带，沉吟聊踯躅。
思为双飞燕，衔泥巢君屋。
```
“四时更变化，岁暮一何速！”，一提到光阴的消逝，往往接着联想到生命的短暂无常。“晨风怀苦心”，“晨风”是《诗经》中《秦风》中的内容，又是古诗词中常见的手法，一个典故、引用，表达了更丰富、更深次的内容，这一句表明了一个才智之士生不逢时，“蟋蟀伤局促”，“蟋蟀”见于《唐风》，这一句想感叹生命的短暂之外，还包含了一层何必如此自苦、不妨及时行乐的意思在内。

下面是《西北有高楼》这首诗。
```
西北有高楼，上与浮云齐。
交疏结绮窗，阿阁三重阶。
上有弦歌声，音响一何悲！
谁能为此曲？无乃杞梁妻。
清商随风发，中曲正徘徊。
一弹再三叹，慷慨有余哀。
不惜歌者苦，但伤知音稀。
愿为双鸿鹄，奋翅起高飞。
```
整首诗有一种对高飞远举的向往追求，也可以体会到一种甘于寂寞、处身高洁的感情和持守。

曹操的诗，在英雄的意志与诗人的才情之中，又结合着一种唯我独尊的霸气。下面是他的《短歌行》。
```
对酒当歌，人生几何！譬如朝露，去日苦多。
慨当以慷，忧思难忘。何以解忧？唯有杜康。
青青子衿，悠悠我心。但为君故，沉吟至今。
呦呦鹿鸣，食野之苹。我有嘉宾，鼓瑟吹笙。
明明如月，何时可掇？忧从中来，不可断绝。
越陌度阡，枉用相存。契阔谈宴，心念旧恩。
月明星稀，乌鹊南飞。绕树三匝，何枝可依？
山不厌高，海不厌深。周公吐哺，天下归心。
```
人们对着歌和酒的时候，精神自然会松弛，感情也就容易激动，平时总是藏在心底的郁闷这时候也会涌上心头。“青青子衿”和“呦呦鹿鸣”两句也是引用了《诗经》中的内容。

曹植是才子型的诗人，纯情善感，缺乏自我反省与节制的能力，诗歌风格随着外界环境的变化而产生明显的变化。早期意气风发，多彩多姿，中期初受压抑，激愤不平，晚期多用象喻抒发心中郁闷。下面是其晚年所做的《七哀》，以弃妇的形象来表达自己与曹丕之间的君臣兄弟关系。
```
明月照高楼，流光正徘徊。
上有愁思妇，悲叹有余哀。
借问叹者谁？自云宕子妻。
君行逾十年，孤妾常独栖。
君若清路尘，妾若浊水泥。
浮沈各异势，会合何时谐？
愿为西南风，长逝入君怀。
君怀良不开，贱妾当何依！
```

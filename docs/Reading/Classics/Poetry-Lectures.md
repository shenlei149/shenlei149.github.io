> 《古诗词课》，叶嘉莹，生活读书新知三联书店，2018 年版，2025 年印刷。

这本书从诗经开始介绍，一直讲到了南宋末年的词人王沂孙。全书分为两个部分，上编为诗，下编为词。共计 36 讲，涉及了几十位诗人和词人，内容丰富，讲解细致。每一讲都介绍了作者的生平背景、创作风格，分析其代表作品，并且结合历史背景和文化背景进行解读。

《诗经》是我国最早的一部诗歌总集。这些诗歌分为风、雅、颂三类，风指的是国风，各地的民间歌谣，雅分为大雅和小雅，除了民间歌谣还有贵族作品，颂是用于宗庙祭祀的诗歌。赋比兴是《诗经》中的三种基本表现手法。赋有铺陈之意，比有拟喻之意，兴有感发兴起之意。
```
苕之华，芸其黄矣。心之忧矣，维其伤矣。
苕之华，其叶青青。知我如此，不如无生。
牂羊坟首，三星在罶。人可以食，鲜可以饱。
```
苕，这里应该念 tiao，二声，指的是一种植物，指凌霄花或水草。到了秋天要凋落的时候就完全变成黄色了。

第一句是见物起兴，属于兴的变现手法。这里是由物及心。心和物之间的感应是微妙的，虽然这种作用之中必然有某种感性的关联，但并非每一种感性的关联都可以做理性的分析。它可以是完全凭直觉的触引，并不一定有理性的思索安排。

牂，zang，一声。牂羊是母羊，坟是大的意思，说明羊很瘦。罶，liu，三声，打鱼的竹筐。水中倒影着星星，说明鱼很少。这里是赋，直接描述。

```
硕鼠硕鼠，无食我黍！三岁贯女，莫我肯顾。
逝将去女，适彼乐土。乐土乐土，爰得我所。

硕鼠硕鼠，无食我麦！三岁贯女，莫我肯德。
逝将去女，适彼乐国。乐国乐国，爰得我直。

硕鼠硕鼠，无食我苗！三岁贯女，莫我肯劳。
逝将去女，适彼乐郊。乐郊乐郊，谁之永号。
```
女同汝。这里用的比的手法。作者心中痛恨那些一味剥削重敛、不顾百姓死活的家伙，不过不能明说，只能用硕鼠来打个比方。比和兴不同，是由心及物，心理已经有了某种情意，找一个外物来做比喻，写出心中的情意。这种表现形式是有理性思索安排的。

```
将仲子兮，无踰我里，无折我树杞。
岂敢爱之，畏我父母。
仲可怀也，父母之言，亦可畏也。

将仲子兮，无踰我墙，无折我树桑。
岂敢爱之，畏我诸兄。
仲可怀也，诸兄之言，亦可畏也。

将仲子兮，无踰我园，无折我树檀。
岂敢爱之，畏人之多言。
仲可怀也，人之多言，亦可畏也。
```
这首诗描述的是女子对所爱男子的叮咛和告诫，直接描述了内心的感情。这种手法就是赋。在传统中，直接描述可以是物，但不仅仅局限于目之所见的具体的外物，也可以是抽象的内心的情感，诗所描述的形象本身就是作者心中的情意。

总结赋比兴三者：赋是直接叙写，即物即心；比是借物为喻，心在物前；兴是因物起兴，物在心前。

《离骚》是《诗经》之后的又一巅峰。作者屈原名平，战国时期的楚国人。司马迁在《史记》中说他因“信而见疑，忠而被谤”，才写了《离骚》抒发自己的情怀，司马迁评价是“推此志也，虽与日月争光可也”。

屈原在《离骚》中反复陈述了希望楚国美好强盛的愿望，在愿望不能实现的苦闷之中，也想过退而自保，独善其身，但是他仍然不肯放弃自己的愿望，写出“既莫足与美政兮，吾将从彭咸之所居！”。屈原所追求的理想是最高远、最完美的，因此也是最难以达到的，但正是由于人类有这样的追求，所以人类才有希望。

中国的诗词，经常表现的一个主题就是对美好的事物、美好的对象、美好的理想的追求和怀思。这个传统，就是从屈原的《离骚》开始的。“美人香草以喻君子”，是中国传统文化中一个独特的比兴方法，源头来自《离骚》。悲秋的主题也是从这里开始传承。

《古诗十九首》是东汉时期一些无名氏的作品，非常易懂又相当难解，艺术成就极高却又没有留下作者的姓名，人人读了心中都觉得有所触动，却又很难说清缘由。《古诗十九首》在写作态度上十分真挚诚恳，语言也相当平易近人，但经受住了千百年来无数读者的反复挖掘，使每个人都够有所得有所感。内容涉及很多人生中常常遇到的问题，写出了人们心灵深处最普通也最深刻的几种感情的基本类型。这些感情来源于人与人以及人与社会之间的关系，凡是社会中的人，都不可能没有经历过。

下面是《东城高且长》这首诗。
```
东城高且长，逶迤自相属。
回风动地起，秋草萋已绿。
四时更变化，岁暮一何速！
晨风怀苦心，蟋蟀伤局促。
荡涤放情志，何为自结束？
燕赵多佳人，美者颜如玉。
被服罗裳衣，当户理清曲。
音响一何悲！弦急知柱促。
驰情整巾带，沉吟聊踯躅。
思为双飞燕，衔泥巢君屋。
```
“四时更变化，岁暮一何速！”，一提到光阴的消逝，往往接着联想到生命的短暂无常。“晨风怀苦心”，“晨风”是《诗经》中《秦风》中的内容，又是古诗词中常见的手法，一个典故、引用，表达了更丰富、更深次的内容，这一句表明了一个才智之士生不逢时，“蟋蟀伤局促”，“蟋蟀”见于《唐风》，这一句想感叹生命的短暂之外，还包含了一层何必如此自苦、不妨及时行乐的意思在内。

下面是《西北有高楼》这首诗。
```
西北有高楼，上与浮云齐。
交疏结绮窗，阿阁三重阶。
上有弦歌声，音响一何悲！
谁能为此曲？无乃杞梁妻。
清商随风发，中曲正徘徊。
一弹再三叹，慷慨有余哀。
不惜歌者苦，但伤知音稀。
愿为双鸿鹄，奋翅起高飞。
```
整首诗有一种对高飞远举的向往追求，也可以体会到一种甘于寂寞、处身高洁的感情和持守。

曹操的诗，在英雄的意志与诗人的才情之中，又结合着一种唯我独尊的霸气。下面是他的《短歌行》。
```
对酒当歌，人生几何！譬如朝露，去日苦多。
慨当以慷，忧思难忘。何以解忧？唯有杜康。
青青子衿，悠悠我心。但为君故，沉吟至今。
呦呦鹿鸣，食野之苹。我有嘉宾，鼓瑟吹笙。
明明如月，何时可掇？忧从中来，不可断绝。
越陌度阡，枉用相存。契阔谈宴，心念旧恩。
月明星稀，乌鹊南飞。绕树三匝，何枝可依？
山不厌高，海不厌深。周公吐哺，天下归心。
```
人们对着歌和酒的时候，精神自然会松弛，感情也就容易激动，平时总是藏在心底的郁闷这时候也会涌上心头。“青青子衿”和“呦呦鹿鸣”两句也是引用了《诗经》中的内容。

曹植是才子型的诗人，纯情善感，缺乏自我反省与节制的能力，诗歌风格随着外界环境的变化而产生明显的变化。早期意气风发，多彩多姿，中期初受压抑，激愤不平，晚期多用象喻抒发心中郁闷。下面是其晚年所做的《七哀》，以弃妇的形象来表达自己与曹丕之间的君臣兄弟关系。
```
明月照高楼，流光正徘徊。
上有愁思妇，悲叹有余哀。
借问叹者谁？自云宕子妻。
君行逾十年，孤妾常独栖。
君若清路尘，妾若浊水泥。
浮沈各异势，会合何时谐？
愿为西南风，长逝入君怀。
君怀良不开，贱妾当何依！
```

阮籍生活在三国时期的魏国，司马家族篡权阶段。他为了保全生命，既不拒绝做官，也不真正干事，平时借酒佯狂，把一切思想和感情都深深地隐藏在心底。下面是他的《咏怀（其一）》。
```
夜中不能寐，起坐弹鸣琴。
薄帷鉴明月，清风吹我襟。
孤鸿号外野，翔鸟鸣北林。
徘徊将何见？忧思独伤心。
```
人和鸟一样，一定要找到一个清白的所在来作为自己安身立命的地方。这个选择往往是痛苦的，有时候很难做到两全其美。最后一句的“见”，并不是指前面那些薄帷、明月、清风和孤鸿，而是指心中所希望看见的东西。这一句还有第二层含义，我知道即将看到什么，那只能是使我更加伤心的悲惨结局。这里，结局就是司马家氏篡权。

嵇康的诗，以“气”与“风神”取胜，下面是他送各个嵇喜从军的诗：《赠秀才入军（其十四）》。
```
息徒兰圃，秣马华山。
流磻平皋，垂竿长川。
目送归鸿，手挥五弦。
俯仰自得，游心太玄。
嘉彼钓叟，得鱼忘筌。
郢人逝矣，谁与尽言？
```
中间四句非常之妙。有的人一辈子总是向外追求的，得到了名誉、地位或财富，内存却没有一点儿真正属于自己的东西。有的人则不是，内心有一个真正属于自己的世界，外界有得有失，有喜有悲，却能超脱世俗之间，能够用一种最自然、最潇洒、最没有虚伪造作的态度去面对外部的世界。

最后四句化用了《庄子》中《外物》篇中的内容，“筌者所以在鱼，得鱼而忘筌”，“言者所以在意，得意而忘言”，这里借用《庄子》，更深次想表达一个事情：我相信我对大自然和人生中那些最本质、最重要的东西已经有所感悟，而且我也不是不肯把这些思想说出来，但是我能够说明白吗？就算我说明白了，听的人并不理解我，他们能够听明白吗？

傅玄，字休奕，晋武帝时期的人。他的诗和同时代的诗略有不同，富于直接感发。首先看下面这首乐府诗《车遥遥篇》。
```
车遥遥兮马洋洋，追思君兮不可忘。
君安游兮西入秦，愿为影兮随君身。
君在阴兮影不见，君依光兮妾所愿。
```
最后两句是说你要是能站在光明之中，那是我心中最大的愿望；你要是站在黑暗之中，我就无法跟你接近了。下面再看看他写的《吴楚歌》。
```
燕人美兮赵女佳，其室则迩兮限曾崖。
云为车兮风为马，玉在山兮兰在野。
云无期兮风有止，思心多端谁能理？
```
兰在野，使人联想到不可改变的本质，那是一种不计酬答的、最美好的本质。玉在山，使人心中怀念的佳人，像玉一样的美好，却又难以相见，不过自屈原开始，这些都是描写君子的说法，这里是说诗人自己。

这里我略过太康和永嘉时期的诗人，直接来看看晋宋之际的谢灵运。谢家是当时的大户人家，不过他从康乐公降为康乐候，食邑三千户减至五百户，为官被排挤，贬职到永嘉。谢灵运的诗歌风格独特，开创了山水诗的先河，拓宽了中国古代诗歌的表现领域。下面是他的《登池上楼》。
```
潜虬媚幽姿，飞鸿响远音。
薄霄愧云浮，栖川怍渊沉。
进德智所拙，退耕力不任。
徇禄及穷海，卧疴对空林。
衾枕昧节候，褰开暂窥临。
倾耳聆波澜，举目眺岖嵚。
初景革绪风，新阳改故阴。
池塘生春草，园柳变鸣禽。
祁祁伤豳歌，萋萋感楚吟。
索居易永久，离群难处心。
持操岂独古，无闷征在今。
```
这首诗并不是由物及心的兴，而是现有一份内心的感慨，通过有心的安排，然后再借助外在景物或古籍来表现出来的。进德智所拙，退耕力不任。这是一种矛盾的心情，做高管没有本事，退耕又没有足够的能力，结果就是徇禄及穷海，卧疴对空林。祁祁伤豳歌，萋萋感楚吟。这两句开始用典，祁祁出自》诗经·国风·豳风·七月》：春日迟迟，采蘩祁祁；女心伤悲，殆及公子同归。萋萋出自《楚辞·招隐士》：王孙游兮不归，春草生兮萋萋。这里传达出谢灵运被贬出首都、远离朝廷，初到人迹罕至的海边，心中感到非常孤独和寂寞的情感。

陶渊明的诗以真淳而论，平淡自然，悠闲自得，同时还包含着抑郁寡欢的悲伤和痛苦，看似平淡的自然诗篇中还蕴含着斑驳曲折的五光十色。最具代表性的就是饮酒，下面是其中的《饮酒（其四）》和《饮酒（其五）》。
```
栖栖失群鸟，日暮犹独飞。
徘徊无定止，夜夜声转悲。
厉响思清远，去来何依依。
因值孤生松，敛翮遥来归。
劲风无荣木，此荫独不衰。
托身已得所，千载不相违。
```
```
结庐在人境，而无车马喧。
问君何能尔？心远地自偏。
采菊东篱下，悠然见南山。
山气日夕佳，飞鸟相与还。
此中有真意，欲辨已忘言。
```
失群鸟，犹独飞。陶渊明宁肯忍受孤独饥寒、流离失所的悲哀，也要追求人生的最高追求：保持一个自尊的真我，实现自身最美好的价值。厉响思清远，去来何依依。他尚未放弃对清远之所的追寻，依然怀着无限无限深情的向往在寻觅一个可以终身寄托的归宿。因值孤生松，敛翮遥来归。这里的“松”是一个象征，那种任凭霜打雪压而依然本色长青的坚贞品质。陶渊明这里追求的是那种真淳质朴、无欺无诈的躬耕生活。此中有真意，欲辨已忘言。他想要辨明着其中的哲理和人生的真谛，可是却想不起用什么语言能够讲清楚，因为这是人生经历中只能感受和体悟到的一种心灵的意会，是人类有了更高层次的理想之后才可能达到的一种精神境界。

初唐是中国诗歌史上格律诗正式形成的时期。所谓格律诗，就是指律诗和绝句。律诗有五言、七言及排律；绝句也有五言和七言两种。格律诗注重诗歌形式上的完美，讲究声调的平仄、诗句的对仗和押韵。比如王勃的《送杜少府之任蜀州》。
```
城阙辅三秦，风烟望五津。
与君离别意，同是宦游人。
海内存知己，天涯若比邻。
无为在歧路，儿女共沾巾。
```
初唐时期的陈子昂是一位有才华的诗人，二十四岁到长安，得到武则天的赏识，他是有政治抱负的人，不过他在长安的政治生涯并不顺利，遭到迫害，最终死在了狱中。下面是《感遇》的其中一首。
```
兰若生夏春，芊蔚何青青。
幽独空林色，朱蕤冒绿蓬。
迟迟白日晚，袅袅秋风生。
岁华尽摇落，芳意竟何成？
```
诗中的兰花与杜若，生长在空寂的山林之中，带着一种默默无闻、安于寂寞、不求人知的境界与情趣。无人欣赏但照样美好。人生一世不过短短几十年的光景，所有的一切，美好的品德、才智、理想、价值又怎样能够实现呢？不得知遇，人生的价值又如何能够实现呢？陈子昂出仕之后，境遇不顺，有了下面这首《感遇》。
```
翡翠巢南海，雄雌珠树林。
何知美人意，骄爱比黄金？
杀身炎州里，委羽玉堂阴。
旖旎光首饰，葳蕤烂锦衾。
岂不在遐远？虞罗忽见寻。
多材信为累，叹息此珍禽。
```
借鸟喻人、自伤身世。这样的知遇比不遇还悲惨。类似陈子昂这样的遭遇，再中国历史上也不在少数。陈子昂最有名的是《登幽州台歌》。
```
前不见古人，后不见来者。
念天地之悠悠，独怆然而涕下！
```
这首诗以感发而取胜，诗人真正掌握了是各种那份最基本的生命之源——强大的生生不息的兴发感动的作用与力量。

唐朝有一类诗被称为山水田园诗，其中代表人物有王维和孟浩然。代表作有王维的《使之塞上》和孟浩然的《过故人庄》。
```
大漠孤烟直，长河落日圆。
萧关逢候骑，都护在燕然。
```
这首诗前面写的自然超妙，后面又出现两句世俗的内容，前后形成了鲜明的对比。这大概是源自王维的自身矛盾，表面上他仕隐两得，内心却是两失的。他既信奉佛教、厌薄尘世，但又干求名利、患得患失；既有丰美深厚的艺术家的才华修养，又难以免除庸俗虚饰的尘世之情。

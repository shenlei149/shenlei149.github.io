> 《古诗词课》，叶嘉莹，生活读书新知三联书店，2018 年版，2025 年印刷。

这本书从诗经开始介绍，一直讲到了南宋末年的词人王沂孙。全书分为两个部分，上编为诗，下编为词。共计 36 讲，涉及了几十位诗人和词人，内容丰富，讲解细致。每一讲都介绍了作者的生平背景、创作风格，分析其代表作品，并且结合历史背景和文化背景进行解读。

《诗经》是我国最早的一部诗歌总集。这些诗歌分为风、雅、颂三类，风指的是国风，各地的民间歌谣，雅分为大雅和小雅，除了民间歌谣还有贵族作品，颂是用于宗庙祭祀的诗歌。赋比兴是《诗经》中的三种基本表现手法。赋有铺陈之意，比有拟喻之意，兴有感发兴起之意。
```
苕之华，芸其黄矣。心之忧矣，维其伤矣。
苕之华，其叶青青。知我如此，不如无生。
牂羊坟首，三星在罶。人可以食，鲜可以饱。
```
苕，这里应该念 tiao，二声，指的是一种植物，指凌霄花或水草。到了秋天要凋落的时候就完全变成黄色了。

第一句是见物起兴，属于兴的变现手法。这里是由物及心。心和物之间的感应是微妙的，虽然这种作用之中必然有某种感性的关联，但并非每一种感性的关联都可以做理性的分析。它可以是完全凭直觉的触引，并不一定有理性的思索安排。

牂，zang，一声。牂羊是母羊，坟是大的意思，说明羊很瘦。罶，liu，三声，打鱼的竹筐。水中倒影着星星，说明鱼很少。这里是赋，直接描述。

```
硕鼠硕鼠，无食我黍！三岁贯女，莫我肯顾。
逝将去女，适彼乐土。乐土乐土，爰得我所。

硕鼠硕鼠，无食我麦！三岁贯女，莫我肯德。
逝将去女，适彼乐国。乐国乐国，爰得我直。

硕鼠硕鼠，无食我苗！三岁贯女，莫我肯劳。
逝将去女，适彼乐郊。乐郊乐郊，谁之永号。
```
女同汝。这里用的比的手法。作者心中痛恨那些一味剥削重敛、不顾百姓死活的家伙，不过不能明说，只能用硕鼠来打个比方。比和兴不同，是由心及物，心理已经有了某种情意，找一个外物来做比喻，写出心中的情意。这种表现形式是有理性思索安排的。

```
将仲子兮，无踰我里，无折我树杞。
岂敢爱之，畏我父母。
仲可怀也，父母之言，亦可畏也。

将仲子兮，无踰我墙，无折我树桑。
岂敢爱之，畏我诸兄。
仲可怀也，诸兄之言，亦可畏也。

将仲子兮，无踰我园，无折我树檀。
岂敢爱之，畏人之多言。
仲可怀也，人之多言，亦可畏也。
```
这首诗描述的是女子对所爱男子的叮咛和告诫，直接描述了内心的感情。这种手法就是赋。在传统中，直接描述可以是物，但不仅仅局限于目之所见的具体的外物，也可以是抽象的内心的情感，诗所描述的形象本身就是作者心中的情意。

总结赋比兴三者：赋是直接叙写，即物即心；比是借物为喻，心在物前；兴是因物起兴，物在心前。

《离骚》是《诗经》之后的又一巅峰。作者屈原名平，战国时期的楚国人。司马迁在《史记》中说他因“信而见疑，忠而被谤”，才写了《离骚》抒发自己的情怀，司马迁评价是“推此志也，虽与日月争光可也”。
